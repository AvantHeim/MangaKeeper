# MangaKeeper
Just for saving manga from readmanga.me

Программа для скачки манги с ридманги и адалтманги.

### Как пользоваться:
* Скачать последнюю версию [здесь](https://github.com/AvantHeim/Readmanga-downloader/releases).
* Поместить в папке с программой файл `input.txt`, в нём разместить ссылку на скачиваемую мангу.
* Запустить программу.
* Ввести краткое название латиницей для именования папок.
* Скачивание идёт "с конца", то есть завершится после скачки главы 1-1.
* Также в папке с программой будет создан файл `<название_манги>.txt`, в котором содержится список глав (с названиями, если они были).

### Пример именования файлов и папок: 
* `Seifuku_Aventure_v01_ch002_[Ley_Line]` - папка 2 главы 1 тома
* `v01c002_001.png` - 1 страница 2 главы 1 тома
